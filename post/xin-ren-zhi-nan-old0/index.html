欢迎来到 OierCraft 服务器，这里是新手指南。

开始游玩

服主是 ReverBer，人称老王。

审核

先加审核群，群号：489364449。

加审核群后会有审核，可以轻松通过，然后进主群并找服主申请白名单（有白名单机制），进入服务器！

进入服务器

刚进入服务器，你会出生在一个木建筑里（不知道该怎么称呼）。这就是位于主城的出生点。

通过水电梯下到一楼，即可进入主城。

理论上主城空余地皮或主城周边区域均可建房，然而由于主城过度开发，钻石等矿物较为缺乏，建议到离主城有一定距离的位置定居。同时，强烈建议在家处设置传送点（home），便于日后回家。关于设置传送点，见“常用命令”。

主城周边有许多地狱传送门，可以直接使用，也可以自建。建议在地狱设置传送点，便于快速前往地狱。

同时你也必须遵守规则。你的行为需要遵守 OIC 法典，否则将会受到相应处罚。

主要插件

本服为彻底的纯净生存，指不添加领地等插件，只保留基础插件，如 tpa 等，方便玩家游玩。

下面是一些重要插件：

  • AdvancedMotd
    用于显示服务器信息，效果如下：

  • Coreprotect
    可以记录玩家的活动，如开箱子或破坏方块，可以查出恶意破坏的玩家。
    所以如果你的个人财产受到损害请及时联系老王帮你查询。

  • Ess 基础插件
    提供服务器基础指令。

  • Skins
    在离线模式下显示皮肤。

  • Tabtps
    用于显示服务器的 Tps 和 Mspt 等。

  • 箱子锁
    将箱子（门、熔炉等）上锁，使得他人无法打开或破坏。
    使用方法：用告示牌右键点击箱子即可上锁。

  • QuickShop
    商店插件，用于玩家之间交易。
    使用方法：
    卖：用你要卖的东西左键点击箱子，在聊天框内输入单价，随后放入物品即可；
    买：左键点击箱子并在聊天框内输入数量即可。
    可以在系统商店卖矿物获得游戏币。

常用命令

  • /tpa <name>
    请求传送到名字为 <name> 的玩家所在的位置。

  • /tpahere <name>
    请求把名字为 <name> 的玩家传送到你所在的位置。

  • /sethome <name>
    将当前所在位置设置为 home，名字为 <name>。

  • /home <name>
    传送到名字为 <name> 的 home。

  • /rmhome <name>
    删除名字为 <name> 的 home。

  • /back
    回到上一次传送的地点。注意无法回到上次死亡地点。

  • /tell <name> <message>
    向名字为 <name> 的玩家私信。

  • /pay <name> <money>
    向名字为 <name> 的玩家转账，转 <money> 游戏币。

  • /tpr
    随机传送。

  • /co i
    CoreProtect 查询命令，输入一次进入查询模式，再输入一次退出。
    在查询模式下，左键点击方块可以查询到方块放置 / 摧毁记录（如果要查询位置没有方块需放置一个），右键点击方块可以查询箱子等的放入 / 取出记录。

关于服务器

服务器官网： https://oiercraft.ga

软件信息

服务器版本为 Java 1.16.5，无正版验证， 基于 Spigot 核心开服。

目前服务器运行在 Windows10 专业版，正在考虑更换 Linux（Centos）。

硬件信息

本服务器运行在华为 Fusion Server RH2288 V2 上，配备了两块 Intel Xeon E5-2620 @2.0Ghz CPU，一共 12 核 24 线程，当然 CPU 主频有点过低，正在考虑更换。

内存条插了 64GB，足够保证服务器稳定运行。

使用 Sakura FRP 进行内网穿透服务，多线路，更稳定。
